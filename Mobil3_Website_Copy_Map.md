## Copy Style Rules (for marketing team)

1. **Never use em dashes or long hyphens.** Use periods. Short sentences.
2. **Never say "ecosistema Web3"** as identity. We are a builder program.
3. **Never sell humo.** No data = don't mention it. Data exists = provide context.
4. **Spanish is primary** for LATAM institutional outreach. English for protocols and global comms.
5. **Words to use:** builders, pipeline, talento, institucional, universidad, adopción real, medible, premium, infraestructura, activación, programas
6. **Words to avoid:** comunidad crypto, ecosistema (as identity), distribution, visibility, awareness, engagement (without metric), disruptivo, revolucionario
7. **Tone:** Direct, confident, non-corporate. YC/Buildspace energy. Says what it does. Shows what it achieved. Invites participation.
8. **All copy changes** go into the page component files. Each ICP page has copy inline using the `t(lang, es, en, pt)` helper. Search for string literals to update.
